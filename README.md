# 🏔️ 山洞逃生 — 张小龙与徐凝柔

> 被追杀者逃入山洞，穿越迷宫，逃出生天

第一人称视角迷宫闯关网页游戏，浏览器打开即玩。

## 玩法

- **WASD** / 虚拟摇杆：移动
- **鼠标拖动**：转向
- **空格/点击**：攻击
- **E**：开箱子/交互
- **F**：传送阵传送
- **1-5**：使用背包道具

## 特色

- 🔦 Raycasting 第一人称3D引擎
- 🌫️ 迷雾战争，有限能见度
- 📦 随机宝箱开出道具
- 👹 5种Boss各有独特技能（减速/中毒/高伤/操作反向/三阶段）
- 🌀 击败Boss激活传送阵
- 🏔️ 5关递增难度
- 📱 手机+PC双端适配
